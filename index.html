About This Website

This website is a simple HTML project created as part of the MERN Stack course Module 2 assignment. It demonstrates the use of fundamental HTML elements including headings, lists, images, tables, forms, and media embeds. The purpose is to showcase basic structuring of a webpage using semantic tags. You’ll find an introduction, image, list of services, a video, price table, and a contact form. This assignment does not use CSS or JavaScript, focusing solely on HTML. I hope this reflects my understanding of web page structure and formatting using pure HTML.
